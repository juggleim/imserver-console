# IMServer Console

IMServer Console 是一个基于 Go 实现的轻量级 IM 管理控制台，提供了面向 Web 的管理界面和 API 网关，用于管理 IM 应用、用户、群组、机器人、消息、推送配置、文件配置、监控与统计数据。

这个项目旨在帮助团队更高效地运维 IM 平台，尤其适合需要统一配置、统一管理和可观测性的生产环境场景。

## 为什么值得关注

- 简化 IM 平台日常运维操作
- 集中管理应用、账号、用户和群组
- 通过结构化管理界面减少手工数据库操作
- 支持推送、RTC、短信、翻译、文件服务等配置管理
- 通过统计、日志和性能指标帮助提升可观测性

## 核心能力

- 账号与应用管理
- 用户、机器人、群组、会话管理
- 消息历史与撤回操作
- 敏感词与拦截器管理
- iOS / Android 推送配置
- 短信、翻译、文件、RTC 配置
- 业务统计、用户活跃度与性能指标
- 内置管理后台 Web UI 与 API 网关

## 技术栈

- Go
- Gin 框架
- GORM
- MySQL
- 基于 Vite 的 Web 前端

## 快速开始

### 前置条件

- Go 1.25 或更高版本
- MySQL 服务
- 可连接的 IM 后端服务

### 配置说明

编辑 [conf/config.yml](conf/config.yml)，设置以下内容：

- 服务端口
- MySQL 连接信息
- IM API 域名和管理域名

### 本地运行

```bash
go run .
```

服务启动后，可通过配置的端口访问管理控制台和相关接口。
